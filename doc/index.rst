Flux CORAL2
===========

This project contains plugins and other add-ons required for Flux to
run as the primary resource manager on HPE Cray EX systems, specifically
the :term:`CORAL-2` systems at Lawrence Livermore National Laboratory:

.. list-table::
   :header-rows: 1

   * - Name
     - Node Count
     - :term:`APU`

   * - `tuolumne <https://hpc.llnl.gov/hardware/compute-platforms/tuolumne>`_
     - 1,152
     - MI-300A

   * - `rzadams <https://hpc.llnl.gov/hardware/compute-platforms/rzadams>`_
     - 128
     - MI-300A

   * - `elcap <https://hpc.llnl.gov/hardware/compute-platforms/el-capitan>`_
     - 11,136
     - MI-300A
